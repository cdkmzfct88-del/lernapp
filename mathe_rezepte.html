1 · Grenzwerte & L'Hôpital
🟠 Wichtig 📊 10–20 Pkt
Wann wird L'Hôpital angewendet?
Nur bei unbestimmten Ausdrücken: 0⁄0 oder ∞⁄∞. Prüfe zuerst direktes Einsetzen — wenn das einen definierten Wert ergibt, fertig.
Schritt 1 Direktes Einsetzen versuchen

Setze den Grenzwert ein. Ergibt sich ein definierter Wert → das ist der Grenzwert. Fertig.

Schritt 2 Unbestimmten Ausdruck identifizieren
0⁄0 oder ∞⁄∞ → L'Hôpital anwendbar
0 · ∞ → umformen zu 0⁄(1⁄∞) oder ∞⁄(1⁄0)
∞ − ∞ → gemeinsamen Nenner suchen
Schritt 3 L'Hôpital anwenden
lim f(x)⁄g(x) = lim f′(x)⁄g′(x)

Zähler und Nenner getrennt ableiten (kein Quotientenregel!). Danach wieder einsetzen.

Schritt 4 Wiederholen falls nötig

Ergibt sich erneut 0⁄0 oder ∞⁄∞ → L'Hôpital ein zweites Mal anwenden.

Schritt 5 Standardgrenzwerte kennen
limx→∞ xⁿ⁄eˣ = 0   (eˣ wächst schneller als jedes Polynom)
limx→∞ ln(x)⁄x = 0
limx→0 sin(x)⁄x = 1
limx→0 (eˣ − 1)⁄x = 1
✦ Vollständiges Beispiel — angelehnt an Klausur 2022

Berechne: limx→0 (eˣ − 1 − x) ⁄ x²

Direktes Einsetzen: (e⁰ − 1 − 0) ⁄ 0² = 0⁄0 → unbestimmt → L'Hôpital
1. Anwendung: f′(x) = eˣ − 1, g′(x) = 2x
→ lim (eˣ − 1) ⁄ (2x)
Einsetzen x = 0: (1 − 1) ⁄ 0 = 0⁄0 → wieder unbestimmt → L'Hôpital nochmal
2. Anwendung: f″(x) = eˣ, g″(x) = 2
→ lim eˣ ⁄ 2
Einsetzen x = 0: e⁰ ⁄ 2 = 1⁄2
Ergebnis: limx→0 (eˣ − 1 − x) ⁄ x² = 1⁄2

Weiteres Beispiel — Polynomgrenzwert:

limx→∞ (3x² + 2x) ⁄ (x² − 5)
= lim (3 + 2⁄x) ⁄ (1 − 5⁄x²)   |÷x²
= (3 + 0) ⁄ (1 − 0) = 3
💡 Klausur-Tipp
  • Bevor du L'Hôpital anwendest: Typ prüfen (0⁄0 oder ∞⁄∞). Ansonsten ist L'Hôpital falsch angewendet.
  • Bei Polynomen durch xhöchste Potenz dividieren — oft schneller als L'Hôpital.
  • Nach jeder L'Hôpital-Anwendung wieder einsetzen — vielleicht ist es schon lösbar.
⚡ Häufige Fehler
  • L'Hôpital auf 1⁄0 anwenden (= ∞, kein unbestimmter Ausdruck) → nicht erlaubt
  • Quotientenregel statt getrennter Ableitung von Zähler und Nenner
  • Vergessen zu prüfen ob nach L'Hôpital nochmals 0⁄0 vorliegt
2 · Partielle Ableitung & Extrema (mehrere Variablen)
🟡 Kann kommen 📊 10–15 Pkt
Schritt 1 Partielle Ableitungen berechnen

Leite nach x ab (y = Konstante), dann nach y (x = Konstante).

fx = ∂f⁄∂x   (y als Konstante behandeln)
fy = ∂f⁄∂y   (x als Konstante behandeln)
Schritt 2 Kritische Punkte: Gleichungssystem lösen
fx(x, y) = 0
fy(x, y) = 0

Löse das System, z.B. durch Einsetzen oder Subtraktion.

Schritt 3 Hesse-Matrix & Determinante
H = ⎡fxx   fxy⎤   det(H) = fxx · fyy − (fxy)²
    ⎣fxy   fyy⎦
Schritt 4 Klassifikation des Extremums
det(H) > 0 und fxx > 0 → lokales Minimum
det(H) > 0 und fxx < 0 → lokales Maximum
det(H) < 0              → Sattelpunkt
det(H) = 0              → keine Aussage möglich
Schritt 5 Funktionswert berechnen

f(x₀, y₀) einsetzen → Koordinaten des Extrempunkts angeben.

✦ Vollständiges Beispiel — angelehnt an Klausur 2025

f(x, y) = x² + y² − 2x − 4y + 6

fx = 2x − 2 = 0  →  x = 1
fy = 2y − 4 = 0  →  y = 2
Kritischer Punkt: (x₀, y₀) = (1, 2)
fxx = 2,   fyy = 2,   fxy = 0
det(H) = 2 · 2 − 0² = 4 > 0  und  fxx = 2 > 0
→ Globales Minimum
f(1, 2) = 1 + 4 − 2 − 8 + 6 = 1
Minimum bei (1 | 2) mit Minimalwert f = 1
💡 Klausur-Tipp Die Hesse-Methode muss vollständig aufgeschrieben werden: alle drei zweiten Ableitungen (fxx, fyy, fxy), dann det(H), dann Klassifikation. Vergisst du auch nur einen Schritt, gibt es Abzug.
⚡ Häufige Fehler
  • fxy = 0 annehmen ohne zu prüfen — immer ausrechnen
  • det(H) > 0 reicht nicht — zusätzlich fxx > 0 oder < 0 bestimmen
  • Sattelpunkt als „kein Extremum" formulieren — korrekt: es ist ein Sattelpunkt
3 · Lagrange-Methode (Optimierung mit Nebenbedingung)
🟡 Kann kommen 📊 10–15 Pkt
Was ist das?
Extremum von f(x, y) suchen, wobei eine Nebenbedingung g(x, y) = 0 eingehalten werden muss.
Schritt 1 Lagrange-Funktion aufstellen
L(x, y, λ) = f(x, y) + λ · g(x, y)
⚠ Vorzeichen der Nebenbedingung

Bringe die NB auf die Form g(x, y) = 0 (alles auf eine Seite). Z.B. aus x + 2y = 10 wird g = 10 − x − 2y oder g = x + 2y − 10.

Schritt 2 Drei partielle Ableitungen gleich null setzen
∂L⁄∂x = 0   (I)
∂L⁄∂y = 0   (II)
∂L⁄∂λ = 0   (III) → das ist wieder die Nebenbedingung!
Schritt 3 System lösen

Aus (I) und (II) λ oder eine Variable eliminieren, dann in (III) einsetzen.

Schritt 4 Ergebnis prüfen und interpretieren

Setze die gefundenen x, y in f(x, y) ein → Extremalwert. Charaktere (Max/Min) per Kontext oder 2nd-order conditions bestimmen.

✦ Vollständiges Beispiel — aus Klausur 2024

Maximiere f(x, y) = x · y   unter der NB   x + 2y = 10

g(x, y) = 10 − x − 2y
L(x, y, λ) = x · y + λ · (10 − x − 2y)
(I)   ∂L⁄∂x = y − λ = 0    →  λ = y
(II)  ∂L⁄∂y = x − 2λ = 0  →  x = 2λ = 2y
(III)  ∂L⁄∂λ = 10 − x − 2y = 0   (NB)

Aus (II): x = 2y. Einsetzen in (III):

10 − 2y − 2y = 0  →  10 = 4y
y* = 2,5  |  x* = 5  |  λ* = 2,5
f(5 ; 2,5) = 5 · 2,5 = 12,5  (Maximum)

Interpretation von λ*: Ein marginaler Anstieg der rechten Seite der NB um 1 Einheit erhöht den Maximalwert um λ* = 2,5 (Schattenpreis).

💡 Klausur-Tipp
  • ∂L⁄∂λ = 0 gibt immer die NB zurück — das ist kein Fehler, das ist der Mechanismus
  • λ immer mitangeben — Fritsch fragt oft nach seiner Interpretation (Schattenpreis)
  • Bei f(x,y) = xy: teile x durch 2 und y durch 1 (proportional zur NB-Koeffizienten)
⚡ Häufige Fehler
  • Vorzeichen in L falsch — NB muss = 0 sein, Vorzeichen von λ anpassen
  • Gleichungssystem falsch aufgelöst — Zwischenschritte aufschreiben
  • λ-Interpretation vergessen — gibt immer Punkte
4 · Gauß-Elimination (Lineares Gleichungssystem)
🔴 Pflicht 📊 15–25 Pkt
Schritt 1 Erweiterte Koeffizientenmatrix [A|b] aufstellen

Schreibe alle Koeffizienten in eine Matrix, rechte Seite (b) als letzte Spalte.

Schritt 2 Vorwärtselimination (Stufenform)

Ziel: unter jeder Pivotposition nur Nullen. Erlaubte Zeilenoperationen:

Ri ← Ri + c · Rj   (Zeile i = Zeile i + c × Zeile j)
Ri ↔ Rj            (Zeilen tauschen)
Ri ← c · Ri         (Zeile skalieren)
Schritt 3 Rückwärtseinsetzen

Starte bei der untersten Zeile → berechne letzte Variable. Dann von unten nach oben einsetzen.

Schritt 4 Probe: Alle Gleichungen prüfen
⚠ Probe IMMER machen

Gibt Punkte und schützt vor Rechenfehlern. Setze alle Lösungen in alle Originalgleichungen ein.

✦ Vollständiges Beispiel — angelehnt an Klausur 2025

Löse das System:

2x + y − z = 8      (I)
−3x − y + 2z = −11  (II)
−2x + y + 2z = −3   (III)

Erweiterte Matrix:

⎡ 2 1 −1 | 8 ⎤ ⎢ −3 −1 2 | −11 ⎥ ⎣ −2 1 2 | −3 ⎦

R₂ ← R₂ + (3⁄2) · R₁:

⎡ 2 1 −1 | 8 ⎤ ⎢ 0 ½ ½ | 1 ⎥ ← −3+3=0, −1+3/2=½, 2−3/2=½, −11+12=1 ⎣ −2 1 2 | −3 ⎦

R₃ ← R₃ + R₁:

⎡ 2 1 −1 | 8 ⎤ ⎢ 0 ½ ½ | 1 ⎥ ⎣ 0 2 1 | 5 ⎦ ← −2+2=0, 1+1=2, 2−1=1, −3+8=5

R₃ ← R₃ − 4 · R₂:

⎡ 2 1 −1 | 8 ⎤ ⎢ 0 ½ ½ | 1 ⎥ ⎣ 0 0 −1 | 1 ⎦ ← 2−4·(½)=0, 1−4·(½)=−1, 5−4·1=1

Rückwärtseinsetzen:

Zeile 3: −z = 1  →  z = −1
Zeile 2: (½)y + (½)(−1) = 1  →  (½)y = 3⁄2  →  y = 3
Zeile 1: 2x + 3 − (−1) = 8  →  2x = 4  →  x = 2
Probe: 2·2 + 3 − (−1) = 4+3+1 = 8 ✓
       −3·2 − 3 + 2·(−1) = −6−3−2 = −11 ✓
       −2·2 + 3 + 2·(−1) = −4+3−2 = −3 ✓
Lösung: x = 2, y = 3, z = −1
💡 Klausur-Tipp Benutze Brüche, nicht Dezimalzahlen — Rechenzeichen bleiben exakt. Schreibe jede Zeilenoperation explizit auf: R₂ ← R₂ + (3⁄2)R₁. Das zeigt Kompetenz und schützt dich, wenn ein Rechenfehler folgt.
⚡ Häufige Fehler
  • Vorzeichen bei negativen Pivots vergessen
  • Zeile nicht vollständig transformieren — auch den b-Wert (rechte Seite) mitrechnen
  • Keine Probe → Flüchtigkeitsfehler werden nicht entdeckt
5 · Matrizenrechnung (Det., Inverse via Adjunkte)
🔴 Pflicht 📊 15–25 Pkt
Schritt 1 Matrizenmultiplikation A · B
Zeile i von A  ×  Spalte j von B  =  (A·B)ij
A(m×n) · B(n×p) → C(m×p)

Bedingung: Spalten von A = Zeilen von B

Schritt 2 Determinante (2×2)
det ⎡a  b⎤ = a·d − b·c
      ⎣c  d⎦
Schritt 3 Determinante (3×3) — Regel von Sarrus
+ a₁₁·a₂₂·a₃₃  + a₁₂·a₂₃·a₃₁  + a₁₃·a₂₁·a₃₂
− a₁₃·a₂₂·a₃₁  − a₁₁·a₂₃·a₃₂  − a₁₂·a₂₁·a₃₃
Schritt 4 Inverse Matrix A⁻¹ (2×2)
A⁻¹ = (1⁄det(A)) · ⎡ d  −b⎤
                     ⎣−c   a⎦

Nur möglich, wenn det(A) ≠ 0. Probe: A · A⁻¹ = I

Schritt 5 LGS mit Inverser lösen: Ax = b  →  x = A⁻¹ · b
✦ Vollständiges Beispiel — aus Klausur 2024

Gegeben: A = ⎡3  1⎤, b = ⎡7⎤

              ⎣5  2⎦       ⎣12⎦

det(A) = 3 · 2 − 1 · 5 = 6 − 5 = 1
A⁻¹ = (1⁄1) · ⎡ 2  −1⎤ = ⎡ 2  −1⎤
             ⎣−5   3⎦    ⎣−5   3⎦
Probe: A · A⁻¹ = ⎡3·2+1·(−5)   3·(−1)+1·3⎤ = ⎡1  0⎤ ✓
               ⎣5·2+2·(−5)   5·(−1)+2·3⎦   ⎣0  1⎦
x = A⁻¹ · b = ⎡2·7 + (−1)·12⎤ = ⎡14−12⎤ = ⎡2⎤
               ⎣(−5)·7 + 3·12⎦   ⎣−35+36⎦   ⎣1⎦
Lösung: x = 2, y = 1
⚡ Häufige Fehler
  • 2×2-Inverse: a und d tauschen, b und c negieren — genau in dieser Reihenfolge
  • Matrizenmult.: A·B ≠ B·A — Reihenfolge beachten
  • Probe vergessen: A·A⁻¹ = I gibt Sicherheit und evtl. Teilpunkte
6 · Simplex-Methode (Lineare Programmierung)
🔴 Pflicht 📊 15–25 Pkt
Standardform: Maximierung, alle NB mit ≤
Schritt 1 Schlupfvariablen einführen

Jede Ungleichung ≤ bekommt eine Schlupfvariable yi ≥ 0:

ax + by ≤ c   wird   ax + by + y₁ = c
Schritt 2 Ausgangstableau aufstellen

Startbasis = alle Schlupfvariablen. Zielfunktionszeile (G-Zeile) mit negativen Koeffizienten der Zielfunktion (Maximierung).

Schritt 3 Pivotspalte wählen (eintretende Variable)

Spalte mit dem betragsmäßig größten negativen Koeffizienten in der G-Zeile.

Schritt 4 Pivotzeile bestimmen — Minimum-Ratio-Test
min(bi ⁄ aij)   für alle   aij > 0

Nur positive Elemente in der Pivotspalte berücksichtigen. Die Zeile mit dem kleinsten Quotienten tritt aus der Basis aus.

Schritt 5 Pivotschritt durchführen

(a) Dividiere die Pivotzeile durch das Pivotelement. (b) Subtrahiere von allen anderen Zeilen das passende Vielfache der neuen Pivotzeile, sodass alle anderen Einträge in der Pivotspalte zu 0 werden.

Schritt 6 Optimalitätsprüfung & Lösung ablesen

Alle Koeffizienten in der G-Zeile ≥ 0 → Optimum erreicht. Sonst: zurück zu Schritt 3.

Basisvariable → Wert = RHS-Eintrag in ihrer Zeile
Nichtbasisvariable → Wert = 0
✦ Vollständiges Beispiel — aus Klausur 2024

Maximiere G = 5K₁ + 4K₂   unter: 2K₁ + K₂ ≤ 8,   K₁ + 2K₂ ≤ 7,   K₁ ≤ 4,   K₁,K₂ ≥ 0

Tableau 1 — Starttableau (Pivotspalte: K₁, Pivotzeile: y₁)

BasisK₁K₂y₁y₂y₃RHS
y₁211008
y₂120107
y₃100014
G−5−40000

Pivotspalte: K₁ (−5, größter Betrag). Ratio-Test: 8⁄2 = 4, 7⁄1 = 7, 4⁄1 = 4. Minimum bei y₁-Zeile → Pivotelement = 2.

Tableau 2 — nach Pivot (K₁ ↔ y₁) (Pivotspalte: K₂, Pivotzeile: y₂)

BasisK₁K₂y₁y₂y₃RHS
K₁1½½004
y₂03⁄2−½103
y₃0−½−½010
G0−3⁄25⁄20020

Pivotspalte: K₂ (−3⁄2). Ratio-Test: 4⁄(½) = 8, 3⁄(3⁄2) = 2 (y₃ überspringen: neg.). Pivot: y₂-Zeile, Element = 3⁄2.

Tableau 3 — Optimal

BasisK₁K₂y₁y₂y₃RHS
K₁102⁄3−1⁄303

Optimum: K₁ = 3, K₂ = 2, Gewinn G = 23.   y₁ = y₂ = 0 (voll ausgelastet), y₃ = 1 (1 Einheit frei).
💡 Klausur-Tipp
  • Ratio-Test: nur positive Elemente berücksichtigen (negativer Nenner → überspringen)
  • Nach jedem Tableau Optimalitätsprüfung — oft vergessen
  • Lösung ablesen: Basisvariablen = RHS-Wert, Nicht-Basisvariablen = 0
  • Schattenpreise: Koeffizienten der Schlupfvariablen in der G-Zeile des Optimaltableaus
⚡ Häufige Fehler
  • Pivotelement falsch identifiziert — erste negative G-Zeilen-Spalte, nicht zwingend die linkeste
  • Pivotzeile: negativen Quotienten mitnehmen → führt zu falschem Ergebnis
  • G-Zeile vergessen zu transformieren — führt zu falschem Zielwert
  • Lösung falsch ablesen: Schlupfvariable in Basis = ihr Wert ≥ 0, nicht 0
7 · Integralrechnung (bestimmtes Integral)
🟠 Wichtig 📊 10–15 Pkt
Stammfunktionen auf einen Blick
∫ xⁿ dx     = xⁿ⁺¹⁄(n+1) + C   (n ≠ −1)
∫ 1⁄x dx   = ln|x| + C
∫ eˣ dx    = eˣ + C
∫ e^(ax) dx = (1⁄a) · e^(ax) + C
∫ aˣ dx    = aˣ ⁄ ln(a) + C
Schritt 1 Stammfunktion F(x) bestimmen

Wende Integrationsregeln an. Integrationskonstante C bei bestimmten Integralen weglassen.

Schritt 2 Bestimmtes Integral berechnen
∫ab f(x) dx = [F(x)]ab = F(b) − F(a)
Schritt 3 Fläche zwischen Kurve und x-Achse
⚠ Vorzeichen bei Flächen

Liegt f(x) < 0 im Intervall, ist das Integral negativ. Für die Fläche: Betrag nehmen. Bei Vorzeichenwechsel in [a,b]: Intervall aufteilen.

✦ Vollständiges Beispiel — aus Klausur 2026

Berechne ∫13 (x² − 2x + 3) dx

F(x) = x³⁄3 − x² + 3x
F(3) = 27⁄3 − 9 + 9 = 9 − 9 + 9 = 9
F(1) = 1⁄3 − 1 + 3 = 1⁄3 + 2 = 7⁄3
∫13 f(x) dx = 9 − 7⁄3 = 27⁄3 − 7⁄3 = 20⁄3 ≈ 6,67

Weitere Aufgabe — Partielle Integration: ∫ x · eˣ dx

u = x,   u′ = 1
v′ = eˣ,   v = eˣ
∫ x · eˣ dx = x · eˣ − ∫ eˣ dx = x · eˣ − eˣ + C = eˣ(x − 1) + C
⚡ Häufige Fehler
  • Hochzahl bei xⁿ⁺¹ falsch — n+1 im Exponenten UND im Nenner
  • F(b) − F(a) in falscher Reihenfolge (nicht F(a) − F(b))
  • Partielle Integration: u und v′ vertauscht wählen — u = Polynom, v′ = eˣ/sin/cos
8 · Finanzmathematik (Renten, Barwert)
🟠 Wichtig 📊 10–15 Pkt
Wichtigste Formeln
Zinseszins:      Kn = K₀ · (1+i)ⁿ
Barwert:         K₀ = Kn ⁄ (1+i)ⁿ
Rentenbarwert:  BW = R · (1 − (1+i)^(−n)) ⁄ i   (nachschüssig)
Rentenendwert:  KW = R · ((1+i)ⁿ − 1) ⁄ i
Annuität:       R = K₀ · i·(1+i)ⁿ ⁄ ((1+i)ⁿ − 1)
Schritt 1 Gegebene Größen identifizieren

Welche Größen sind gegeben? K₀, Kn, R, i, n — was soll berechnet werden?

Schritt 2 Formel wählen und einsetzen

Nachschüssig = Zahlung am Ende jeder Periode. Vorschüssig = Zahlung am Anfang → Formeln unterscheiden sich.

Schritt 3 Ergebnis interpretieren

Barwert = heutiger Wert aller zukünftigen Zahlungen bei Zinssatz i. Annuität = gleichbleibende Zahlung über n Perioden.

✦ Vollständiges Beispiel — aus Klausur 2025

Rente: R = 1.000 €⁄Jahr, n = 5 Jahre, i = 5 %, nachschüssig. Berechne Barwert.

BW = 1.000 · (1 − (1,05)^(−5)) ⁄ 0,05

(1,05)⁵ = 1,27628  →  (1,05)^(−5) = 1⁄1,27628 = 0,78353

BW = 1.000 · (1 − 0,78353) ⁄ 0,05
     = 1.000 · 0,21647 ⁄ 0,05
     = 1.000 · 4,3295
Barwert = 4.329,48 €

Annuität: Kredit K₀ = 10.000 €, n = 4 Jahre, i = 6 %

R = 10.000 · 0,06 · (1,06)⁴ ⁄ ((1,06)⁴ − 1)
  = 10.000 · 0,06 · 1,26248 ⁄ (1,26248 − 1)
  = 10.000 · 0,07575 ⁄ 0,26248
  = 10.000 · 0,28859
Annuität = 2.885,91 € ⁄ Jahr
⚡ Häufige Fehler
  • Zinssatz als Prozent statt Dezimalzahl einsetzen: i = 0,05 nicht 5
  • Vor- vs. nachschüssig verwechseln — lese die Aufgabe zweimal
  • (1+i)^(−n) ohne Taschenrechner falsch berechnen — Zwischenstand aufschreiben
9 · Summen- & Produktnotation + Ungleichungen
🟠 Wichtig 📊 5–10 Pkt
Warum wichtig?
Erscheint regelmäßig als Aufwärmaufgabe in Fritsch-Klausuren — günstige Punkte, die mit etwas Übung sicher geholt werden können. Ungleichungen kamen in 2⁄3 Klausuren vor.
Summen- und Produktnotation
Summe:    Σi=1n ai = a₁ + a₂ + … + an
Produkt:  Πi=1n ai = a₁ · a₂ · … · an

Rechenregeln:
Σ(ai + bi) = Σai + Σbi
Σ c · ai    = c · Σai   (Konstante ausklammern)
Σi=1n 1    = n
Σi=1n i    = n(n+1)⁄2
Σi=1n i²   = n(n+1)(2n+1)⁄6
Ungleichungen lösen
Wie Gleichung, AUSSER:
Bei Multiplikation⁄Division mit negativer Zahl → Ungleichungszeichen umdrehen!
−2x > 6  →  x < −3   (Zeichen dreht sich!)
Schritt 1 Summenausdruck konkret ausschreiben

Schreibe die ersten 3 Glieder explizit hin — hilft beim Verstehen und zeigt dem Prüfer, dass du die Notation kennst.

Schritt 2 Konstante ausklammern / umformen

Konstanten (unabhängig von i) vor das Summenzeichen ziehen.

Schritt 3 Standardformel anwenden oder aufsummieren
