1 · Ableitungsregeln & Kurvendiskussion
🔴 Pflicht 📊 20–30 Pkt
Ableitungsregeln auf einen Blick
Potenzregel: (xⁿ)' = n·xⁿ⁻¹ Produktregel: (u·v)' = u'·v + u·v' Quotientenregel:(u/v)' = (u'·v − u·v') / v² Kettenregel: (f(g(x)))' = f'(g(x)) · g'(x) ln(x): (ln x)' = 1/x eˣ: (eˣ)' = eˣ eᵍ⁽ˣ⁾: (e^g(x))' = g'(x)·e^g(x)
Schritt-für-Schritt Kurvendiskussion
1

Definitionsbereich

Ausschließen: Division durch 0, negativer Radikand, ln(0). Meist: D = ℝ oder D = ℝ \ {0}

2

Nullstellen: f(x) = 0 lösen

Ausklammern, PQ-Formel, oder numerisch schätzen.

3

Erste Ableitung f'(x)

Berechne f'(x) mit den Ableitungsregeln.

4

Kritische Punkte: f'(x) = 0 lösen

Jede Lösung x₀ ist ein Kandidat für Extremstellen.

5

Zweite Ableitung f''(x) → Extremum prüfen

f''(x₀) > 0 → lokales Minimum f''(x₀) < 0 → lokales Maximum f''(x₀) = 0 → weitere Untersuchung nötig (Vorzeichenwechsel f')
6

Wendepunkte: f''(x) = 0 und Vorzeichenwechsel

f''(x₀) = 0 UND f'''(x₀) ≠ 0 → Wendepunkt bei x₀
7

Funktionswert berechnen und Ergebnis formulieren

Setze x₀ in f(x) ein → Koordinaten des Extrempunkts (x₀ | f(x₀))

✦ Vollständiges Beispiel (aus Klausur 2025)

f(x) = −2x³ + 12x² − 18x + 5

f'(x) = −6x² + 24x − 18

Kritische Punkte: f'(x) = 0

−6x² + 24x − 18 = 0 |÷(−6) x² − 4x + 3 = 0 (x−1)(x−3) = 0 → x₁ = 1, x₂ = 3
f''(x) = −12x + 24 f''(1) = −12 + 24 = +12 > 0 → lokales Minimum f''(3) = −36 + 24 = −12 < 0 → lokales Maximum
f(1) = −2 + 12 − 18 + 5 = −3 → Tiefpunkt T(1|−3) f(3) = −54 + 108 − 54 + 5 = 5 → Hochpunkt H(3|5)
💡 Klausur-Tipp
  • Immer alle Schritte aufschreiben, auch wenn du das Ergebnis siehst — Punkte gibt es für den Rechenweg.
  • Kettenregel: Außenableitung mal Innenableitung — in genau dieser Reihenfolge aufschreiben.
  • f''(x₀) = 0 reicht NICHT für Wendepunkt — Vorzeichenwechsel prüfen!

⚡ Häufige Fehler

  • Produktregel vergessen: (x·eˣ)' ≠ eˣ; korrekt: eˣ + x·eˣ = eˣ(1+x)
  • f''(x₀) = 0 als Wendepunkt werten ohne Vorzeichenwechsel zu prüfen
  • Koordinaten des Extremums: auch f(x₀) berechnen, nicht nur x₀ angeben
2 · Partielle Ableitung & Extrema (mehrere Variablen)
🔴 Pflicht 📊 20–30 Pkt
1

Partielle Ableitungen berechnen

Leite nach x ab (y = Konstante), dann nach y (x = Konstante).

fₓ = ∂f/∂x (y als Konstante behandeln) fᵧ = ∂f/∂y (x als Konstante behandeln)
2

Kritische Punkte: Gleichungssystem lösen

fₓ(x, y) = 0 fᵧ(x, y) = 0

Löse das System, z.B. durch Einsetzen oder Subtraktion.

3

Hesse-Matrix & Determinante

H = [[fₓₓ, fₓᵧ], det(H) = fₓₓ·fᵧᵧ − (fₓᵧ)² [fₓᵧ, fᵧᵧ]]
4

Klassifikation des Extremums

det(H) > 0 UND fₓₓ > 0 → lokales Minimum det(H) > 0 UND fₓₓ < 0 → lokales Maximum det(H) < 0 → Sattelpunkt det(H) = 0 → keine Aussage möglich
5

Funktionswert berechnen

f(x₀, y₀) einsetzen → Koordinaten des Extrempunkts angeben.

✦ Vollständiges Beispiel (angelehnt an Klausur 2025)

f(x, y) = x² + y² − 2x − 4y + 6

fₓ = 2x − 2 = 0 → x = 1 fᵧ = 2y − 4 = 0 → y = 2 Kritischer Punkt: (x₀, y₀) = (1, 2)
fₓₓ = 2, fᵧᵧ = 2, fₓᵧ = 0 det(H) = 2·2 − 0² = 4 > 0 und fₓₓ = 2 > 0 → Globales Minimum
f(1, 2) = 1 + 4 − 2 − 8 + 6 = 1 Minimum bei (1 | 2) mit Minimalwert 1
💡 Klausur-Tipp
Die Hesse-Methode muss vollständig aufgeschrieben werden: alle drei zweiten Ableitungen (fₓₓ, fᵧᵧ, fₓᵧ), dann det(H), dann Klassifikation. Vergisst du auch nur einen Schritt, gibt es Abzug.

⚡ Häufige Fehler

  • fₓᵧ = 0 annehmen ohne zu prüfen — immer ausrechnen
  • det(H) > 0 reicht nicht — zusätzlich fₓₓ > 0 oder < 0 bestimmen
  • Sattelpunkt als „kein Extremum" formulieren — korrekt: es ist ein Sattelpunkt
3 · Lagrange-Methode (Optimierung mit Nebenbedingung)
🔴 Pflicht 📊 15–25 Pkt
Was ist das?

Extremum von f(x, y) suchen, wobei eine Nebenbedingung g(x, y) = 0 eingehalten werden muss.

1

Lagrange-Funktion aufstellen

L(x, y, λ) = f(x, y) + λ · g(x, y)
⚠ Vorzeichen der Nebenbedingung
Bringe die NB auf die Form g(x, y) = 0 (alles auf eine Seite). Z.B. aus x + 2y = 10 wird g = 10 − x − 2y oder g = x + 2y − 10.
2

Drei partielle Ableitungen gleich null setzen

∂L/∂x = 0 (I) ∂L/∂y = 0 (II) ∂L/∂λ = 0 (III) → das ist wieder die Nebenbedingung!
3

System lösen

Aus (I) und (II) λ oder eine Variable eliminieren, dann in (III) einsetzen.

4

Ergebnis prüfen und interpretieren

Setze die gefundenen x, y in f(x, y) ein → Extremalwert. Charaktere (Max/Min) per Kontext oder 2nd-order conditions bestimmen.

✦ Vollständiges Beispiel (aus Klausur 2024)

Maximiere f(x, y) = x·y   unter der NB   x + 2y = 10

g(x, y) = 10 − x − 2y L(x, y, λ) = x·y + λ·(10 − x − 2y)
(I) ∂L/∂x = y − λ = 0 → λ = y (II) ∂L/∂y = x − 2λ = 0 → x = 2λ = 2y (III) ∂L/∂λ = 10 − x − 2y = 0 (NB)

Aus (II): x = 2y. Einsetzen in (III):

10 − 2y − 2y = 0 10 = 4y y* = 2,5 x* = 2·2,5 = 5 λ* = y* = 2,5
f(5; 2,5) = 5 · 2,5 = 12,5 (Maximum)

Interpretation: Bei x = 5 und y = 2,5 ist das Maximum unter der gegebenen Budgetrestriktion erreicht.

Interpretation von λ*: Ein marginaler Anstieg der rechten Seite der NB um 1 Einheit erhöht den Maximalwert um λ* = 2,5. (Schattenpreis)

💡 Klausur-Tipp
  • ∂L/∂λ = 0 gibt immer die NB zurück — das ist kein Fehler, das ist der Mechanismus
  • λ immer mitangeben — Fritsch fragt oft nach seiner Interpretation (Schattenpreis)
  • Bei f(x,y) = xy: teile x durch 2 und y durch 1 (proportional zur NB-Koeffizienten)

⚡ Häufige Fehler

  • Vorzeichen in L falsch — NB muss = 0 sein, Vorzeichen von λ anpassen
  • Gleichungssystem falsch aufgelöst — Zwischenschritte aufschreiben
  • λ-Interpretation vergessen — gibt immer Punkte
4 · Gauß-Elimination (Lineares Gleichungssystem)
🔴 Pflicht 📊 15–25 Pkt
1

Erweiterte Koeffizientenmatrix [A|b] aufstellen

Schreibe alle Koeffizienten in eine Matrix, rechte Seite (b) als letzte Spalte.

2

Vorwärtselimination (Stufenform)

Ziel: unter jeder Pivotposition nur Nullen. Erlaubte Zeilenoperationen:

Rᵢ ← Rᵢ + c · Rⱼ (Zeile i = Zeile i + c × Zeile j) Rᵢ ↔ Rⱼ (Zeilen tauschen) Rᵢ ← c · Rᵢ (Zeile skalieren)
3

Rückwärtseinsetzen

Starte bei der untersten Zeile → berechne letzte Variable. Dann von unten nach oben einsetzen.

4

Probe: Alle Gleichungen prüfen

⚠ Probe IMMER machen
Gibt Punkte und schützt vor Rechenfehlern. Setze alle Lösungen in alle Originalgleichungen ein.
✦ Vollständiges Beispiel (angelehnt an Klausur 2025)

Löse das System:

2x + y − z = 8 (I) −3x − y + 2z = −11 (II) −2x + y + 2z = −3 (III)

Erweiterte Matrix:

[ 2 1 −1 | 8] [−3 −1 2 | −11] [−2 1 2 | −3 ]

R2 ← R2 + (3/2)·R1:

[ 2 1 −1 | 8] [ 0 1/2 1/2 | 1] ← −3+3=0, −1+3/2=1/2, 2−3/2=1/2, −11+12=1 [−2 1 2 | −3]

R3 ← R3 + R1:

[ 2 1 −1 | 8] [ 0 1/2 1/2 | 1] [ 0 2 1 | 5] ← −2+2=0, 1+1=2, 2−1=1, −3+8=5

R3 ← R3 − 4·R2:

[ 2 1 −1 | 8] [ 0 1/2 1/2 | 1] [ 0 0 −1 | 1] ← 2−4·(1/2)=0, 1−4·(1/2)=−1, 5−4·1=1

Rückwärtseinsetzen:

Zeile 3: −z = 1 → z = −1 Zeile 2: (1/2)y + (1/2)·(−1) = 1 (1/2)y = 3/2 → y = 3 Zeile 1: 2x + 3 − (−1) = 8 2x = 4 → x = 2
Probe: 2·2 + 3 − (−1) = 4+3+1 = 8 ✓ −3·2 − 3 + 2·(−1) = −6−3−2 = −11 ✓ −2·2 + 3 + 2·(−1) = −4+3−2 = −3 ✓
💡 Klausur-Tipp
Benutze Brüche, nicht Dezimalzahlen — Rechenzeichen bleiben exakt. Schreibe jede Zeilenoperation explizit auf: R2 ← R2 + (3/2)R1. Das zeigt Kompetenz und schützt dich, wenn ein Rechenfehler folgt.

⚡ Häufige Fehler

  • Vorzeichen bei negativen Pivots vergessen
  • Zeile nicht vollständig transformieren — auch den b-Wert (rechte Seite) mitrechnen
  • Keine Probe → Flüchtigkeitsfehler werden nicht entdeckt
5 · Matrizenrechnung (Mult., Det., Inverse)
🟠 Wichtig 📊 15–20 Pkt
1

Matrizenmultiplikation A · B

Zeile i von A × Spalte j von B = (A·B)ᵢⱼ A(m×n) · B(n×p) → C(m×p)

Bedingung: Spalten von A = Zeilen von B

2

Determinante (2×2)

det [[a, b], [c, d]] = a·d − b·c
3

Determinante (3×3) — Regel von Sarrus

+ a₁₁·a₂₂·a₃₃ + a₁₂·a₂₃·a₃₁ + a₁₃·a₂₁·a₃₂ − a₁₃·a₂₂·a₃₁ − a₁₁·a₂₃·a₃₂ − a₁₂·a₂₁·a₃₃
4

Inverse Matrix A⁻¹ (2×2)

A⁻¹ = (1/det(A)) · [[d, −b], [−c, a]]

Nur möglich, wenn det(A) ≠ 0. Probe: A·A⁻¹ = I

5

LGS mit Inverser lösen: Ax = b → x = A⁻¹·b

✦ Vollständiges Beispiel (aus Klausur 2024)

Gegeben: A = [[3, 1], [5, 2]], b = [[7], [12]]

det(A) = 3·2 − 1·5 = 6 − 5 = 1
A⁻¹ = (1/1) · [[ 2, −1], [−5, 3]] = [[2, −1], [−5, 3]]
Probe: A · A⁻¹ = [[3·2+1·(−5), 3·(−1)+1·3], [5·2+2·(−5), 5·(−1)+2·3]] = [[1, 0], [0, 1]] ✓
x = A⁻¹ · b = [[2·7 + (−1)·12], [(−5)·7 + 3·12]] = [[14−12], [−35+36]] = [[2], [1]] → x = 2, y = 1

⚡ Häufige Fehler

  • 2×2-Inverse: a und d tauschen, b und c negieren — genau in dieser Reihenfolge
  • Matrizenmult.: A·B ≠ B·A — Reihenfolge beachten
  • Probe vergessen: A·A⁻¹ = I gibt Sicherheit und evtl. Teilpunkte
6 · Simplex-Methode (Lineare Programmierung)
🟠 Wichtig 📊 15–25 Pkt
Standardform: Maximierung, alle NB mit ≤
1

Schlupfvariablen einführen

Jede Ungleichung ≤ bekommt eine Schlupfvariable yᵢ ≥ 0: ax + by ≤ c wird ax + by + y₁ = c.

2

Ausgangstableau aufstellen

Startbasis = alle Schlupfvariablen. Zielfunktionszeile mit negativen Koeffizienten der Zielfunktion (Maximierung).

3

Pivotspalte (= Eintretende Variable)

Spalte mit dem betragsmäßig größten negativen Koeffizienten in der Zielfunktionszeile.

4

Pivotzeile (= Austretende Variable)

Minimum-Ratio-Test: min(bᵢ / aᵢⱼ) für aᵢⱼ > 0

Nur positive Elemente in der Pivotspalte berechnen.

5

Pivotschritt

(a) Teile Pivotzeile durch Pivotelement. (b) Alle anderen Zeilen: Subtrahiere passendes Vielfaches der neuen Pivotzeile.

6

Optimalitätsprüfung

Alle Koeffizienten in der Zielfunktionszeile ≥ 0 → optimal! Sonst: Schritt 3 wiederholen.

✦ Vollständiges Beispiel (aus Klausur 2024)

Maximiere G = 5K₁ + 4K₂ s.t.: 2K₁+K₂ ≤ 8 · K₁+2K₂ ≤ 7 · K₁ ≤ 4 · K₁,K₂ ≥ 0

Tableau 1 (Start):

BasisK₁K₂y₁y₂y₃RHS
y₁211008
y₂120107
y₃100014
G−5−40000

Pivotspalte: K₁ (−5 ist betragsgrößte negative Zahl). Ratio-Test: 8/2=4, 7/1=7, 4/1=4. Pivot: y₁-Zeile, Element = 2. → K₁ tritt ein, y₁ tritt aus.

Tableau 2 (nach Pivot K₁ ↔ y₁):

BasisK₁K₂y₁y₂y₃RHS
K₁1½½004
y₂03/2−½103
y₃0−½−½010
G0−3/25/20020

Pivotspalte: K₂ (−3/2). Ratio-Test: 4/(½)=8, 3/(3/2)=2, überspringe (neg.). Pivot: y₂-Zeile, Element = 3/2.

Tableau 3 (Optimal):

BasisK₁K₂y₁y₂y₃RHS
K₁102/3−1/303
K₂01−1/32/302
y₃00−2/31/311
G0021023

Alle G-Zeilen-Koeffizienten ≥ 0 → Optimum: K₁ = 3, K₂ = 2, G = 23

💡 Klausur-Tipp
  • Ratio-Test: nur positive Elemente berücksichtigen (negativer Nenner → überspringen)
  • Nach jedem Tableau Optimalitätsprüfung — oft vergessen
  • Lösung aus dem finalen Tableau ablesen: Basisvariablen = RHS-Wert, Nicht-Basisvariablen = 0

⚡ Häufige Fehler

  • Pivotelement falsch identifiziert — erste negative Zielfunktionsspalte, nicht zwingend linkeste
  • Pivotzeile: negativen Quotienten mitnehmen → führt zu falschem Ergebnis
  • G-Zeile vergessen zu transformieren — führt zu falschem Zielwert
  • Lösung falsch ablesen: Schlupfvariable in Basis = ihr Wert, nicht 0
7 · Integralrechnung (bestimmtes Integral)
🟠 Wichtig 📊 10–15 Pkt
Stammfunktionen auf einen Blick
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C (n ≠ −1) ∫ 1/x dx = ln|x| + C ∫ eˣ dx = eˣ + C ∫ e^(ax) dx = (1/a)·e^(ax) + C ∫ a^x dx = a^x / ln(a) + C
1

Stammfunktion F(x) bestimmen

Wende Integrationsregeln an. Integrationskonstante C bei bestimmten Integralen weglassen.

2

Bestimmtes Integral berechnen

∫ₐᵇ f(x) dx = [F(x)]ₐᵇ = F(b) − F(a)
3

Fläche zwischen Kurve und x-Achse

⚠ Vorzeichen bei Flächen
Liegt f(x) < 0 im Intervall, ist das Integral negativ. Für die Fläche: Betrag nehmen. Bei Vorzeichenwechsel in [a,b]: Intervall aufteilen.
✦ Vollständiges Beispiel (aus Klausur 2026)

Berechne ∫₁³ (x² − 2x + 3) dx

F(x) = x³/3 − x² + 3x
F(3) = 27/3 − 9 + 9 = 9 − 9 + 9 = 9 F(1) = 1/3 − 1 + 3 = 1/3 + 2 = 7/3
∫₁³ f(x) dx = F(3) − F(1) = 9 − 7/3 = 27/3 − 7/3 = 20/3 ≈ 6,67

Weitere Aufgabe — Partielle Integration: ∫ x·eˣ dx

u = x, u' = 1 v' = eˣ, v = eˣ ∫ x·eˣ dx = x·eˣ − ∫ eˣ dx = x·eˣ − eˣ + C = eˣ(x−1) + C

⚡ Häufige Fehler

  • Hochzahl bei xⁿ⁺¹ falsch — n+1 im Exponenten UND im Nenner
  • F(b) − F(a) in falscher Reihenfolge (nicht F(a) − F(b))
  • Partielle Integration: u und v' vertauscht wählen — u = Polynom, v' = eˣ/sin/cos
8 · Finanzmathematik (Renten, Barwert)
🟡 Kann kommen 📊 10–15 Pkt
Wichtigste Formeln
Zinseszins: Kₙ = K₀ · (1+i)ⁿ Barwert: K₀ = Kₙ / (1+i)ⁿ Rentenbarwert: BW = R · (1 − (1+i)^(−n)) / i (nachschüssig) Rentenendwert: KW = R · ((1+i)ⁿ − 1) / i Annuität: R = K₀ · i·(1+i)ⁿ / ((1+i)ⁿ − 1)
1

Gegebene Größen identifizieren

Welche Größen sind gegeben? K₀, Kₙ, R, i, n — was soll berechnet werden?

2

Formel wählen und einsetzen

Nachschüssig = Zahlung am Ende jeder Periode. Vorschüssig = Zahlung am Anfang → Formeln unterscheiden sich.

3

Ergebnis interpretieren

Barwert = heutiger Wert aller zukünftigen Zahlungen bei Zinssatz i. Annuität = gleichbleibende Zahlung über n Perioden.

✦ Vollständiges Beispiel (aus Klausur 2025)

Rente: R = 1.000 €/Jahr, n = 5 Jahre, i = 5%, nachschüssig. Berechne Barwert.

BW = 1.000 · (1 − (1,05)^(−5)) / 0,05 (1,05)^5 = 1,27628 → (1,05)^(−5) = 1/1,27628 = 0,78353 BW = 1.000 · (1 − 0,78353) / 0,05 = 1.000 · 0,21647 / 0,05 = 1.000 · 4,3295 = 4.329,48 €

Annuität: Kredit K₀ = 10.000 €, n = 4 Jahre, i = 6%

R = 10.000 · 0,06·(1,06)⁴ / ((1,06)⁴ − 1) = 10.000 · 0,06·1,26248 / (1,26248 − 1) = 10.000 · 0,07575 / 0,26248 = 10.000 · 0,28859 = 2.885,91 €

⚡ Häufige Fehler

  • Zinssatz als Prozent statt Dezimalzahl einsetzen: i = 0,05 nicht 5
  • Vor- vs. nachschüssig verwechseln — lese die Aufgabe zweimal
  • (1+i)^(-n) ohne Taschenrechner falsch berechnen — Zwischenstand aufschreiben
9 · Folgen und Reihen (geom. / arithm.)
🟡 Kann kommen 📊 10–15 Pkt
Formeln
Arithmetische Folge: aₙ = a₁ + (n−1)·d Arithmetische Reihe: Sₙ = n·(a₁+aₙ)/2 = n·(2a₁+(n−1)d)/2 Geometrische Folge: aₙ = a₁ · qⁿ⁻¹ Geometrische Reihe: Sₙ = a₁·(qⁿ−1)/(q−1) (q ≠ 1) Unendliche Reihe: S∞ = a₁/(1−q) (|q| < 1)
1

Typ bestimmen: arithmetisch oder geometrisch?

Arithmetisch: konstante Differenz d = aₙ₊₁ − aₙ. Geometrisch: konstantes Verhältnis q = aₙ₊₁/aₙ.

2

Parameter bestimmen: a₁, d oder q

Entweder direkt gegeben oder aus zwei Gliedern ausrechnen.

3

Formel anwenden und Ergebnis interpretieren

✦ Vollständiges Beispiel (angelehnt an Klausur 2024)

Geometrisch: a₁ = 100, q = 0,9. Berechne a₅ und S₅.

a₅ = 100 · 0,9⁴ = 100 · 0,6561 = 65,61 S₅ = 100 · (0,9⁵ − 1)/(0,9 − 1) = 100 · (0,59049−1)/(−0,1) = 100 · (−0,40951)/(−0,1) = 100 · 4,0951 = 409,51

Unendliche Summe (|q|=0,9 < 1):

S∞ = 100 / (1−0,9) = 100 / 0,1 = 1.000

Arithmetisch: a₁ = 3, d = 5. Welches Glied hat den Wert 48?

aₙ = 3 + (n−1)·5 = 48 (n−1)·5 = 45 → n−1 = 9 → n = 10

⚡ Häufige Fehler

  • Exponent bei aₙ: ist n−1, nicht n
  • Geometrische Reihe mit q = 1: Formel nicht anwendbar → einfach n·a₁
  • S∞ nur bei |q| < 1 möglich — sonst divergiert die Reihe

📋 Formelreferenz — alle Formeln auf einen Blick

Ableitungsregeln

(xⁿ)' = n·xⁿ⁻¹
(u·v)' = u'v + uv'
(f(g))' = f'(g)·g'

Hesse-Matrix

det(H) = fₓₓ·fᵧᵧ − fₓᵧ²
det>0, fₓₓ>0 → Min
det>0, fₓₓ<0 → Max

Lagrange

L = f + λ·g
∂L/∂x=∂L/∂y=∂L/∂λ=0

Inverse 2×2

A⁻¹ = (1/det)·[[d,−b],[−c,a]]
det = ad − bc

Simplex

Pivotspalte: min G-Koeff.
Pivotzeile: min(b/a), a>0

Integrale

∫xⁿ = xⁿ⁺¹/(n+1)
∫eˣ = eˣ
∫₁ᵇ f = F(b)−F(a)
