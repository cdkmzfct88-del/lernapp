1 · KQ-Schätzung (Lineare Regression)
🔴 Pflicht 📊 20–40 Pkt
Was ist das?

Die Methode der kleinsten Quadrate (KQ) findet die Gerade Ŷ = α̂ + β̂X, die die Summe der quadrierten Abweichungen Σε̂ᵢ² minimiert.

Wann verwenden?
  • Zusammenhang zwischen zwei metrischen Variablen analysieren
  • Steigung und Achsenabschnitt einer Regressionsgerade bestimmen
  • Bestimmtheitsmaß R² berechnen
Schritt-für-Schritt
1

Tabelle aufstellen

Lege Spalten an für: i | xᵢ | yᵢ | xᵢ² | xᵢyᵢ | ŷᵢ | ε̂ᵢ | ε̂ᵢ² — fülle xᵢ und yᵢ aus den Daten, alles andere berechnest du.

2

Summen berechnen

Σxᵢ, Σyᵢ, Σxᵢ², Σxᵢyᵢ

Addiere jede Spalte einzeln. Kontrolliere durch gegenseitiges Nachrechnen.

3

Mittelwerte berechnen

x̄ = Σxᵢ / n     ȳ = Σyᵢ / n
4

β̂ berechnen

Hauptformel (für Tabellenwerte):
β̂ = (Σxᵢyᵢ − n·x̄·ȳ) / (Σxᵢ² − n·x̄²)
Alternative (konzeptuell klarer):
β̂ = Σ(xᵢ−x̄)(yᵢ−ȳ) / Σ(xᵢ−x̄)²

Die erste Formel ist einfacher, wenn du die Summen schon berechnet hast.

5

α̂ berechnen

α̂ = ȳ − β̂ · x̄

Die Regressionsgerade läuft immer durch den Punkt (x̄, ȳ).

6

Regressionsgerade aufschreiben

Ŷ = α̂ + β̂ · X

Setze die berechneten Werte ein. Schreibe die Gleichung vollständig mit Zahlen auf.

7

Bestimmtheitsmaß R² berechnen

R² = 1 − Σε̂ᵢ² / Σ(yᵢ−ȳ)²

Berechne erst die Residuen ε̂ᵢ = yᵢ − ŷᵢ, dann ε̂ᵢ² und summiere. R² ∈ [0, 1].

⚠ Interpretation Pflicht
„R² = 0,95 bedeutet: 95 % der Varianz von Y wird durch X erklärt."
8

β̂ interpretieren

„Steigt X um 1 Einheit, steigt Y durchschnittlich um β̂ Einheiten."
Einheiten immer benennen (z.B. „cm", „kg", „€").

✦ Vollständiges Beispiel (aus Klausur 2026)

Daten: n = 5 Beobachtungen zum Zusammenhang Werbungsausgaben (x) und Umsatz (y).

💡 Klausur-Tipp
Schnurbus fragt fast immer: (1) Tabelle ausfüllen, (2) Schätzer berechnen + interpretieren, (3) Bestimmtheitsmaß. Lerne diesen Ablauf auswendig. Die Tabelle ist das Herzstück — mach sie sorgfältig.

⚡ Häufige Fehler

  • α̂ und β̂ vertauschen (β̂ = Steigung, α̂ = Achsenabschnitt)
  • Vorzeichen bei ε̂ᵢ = yᵢ − ŷᵢ vergessen (nicht ŷᵢ − yᵢ)
  • Interpretation von β̂ weglassen — immer Einheiten nennen
  • R² ohne Interpretation lassen — 1 Satz reicht, gibt Punkte
2 · Hypothesentest für Regressionsparameter
🔴 Pflicht 📊 15–25 Pkt
Was ist das?

Prüft, ob der geschätzte Parameter β̂ statistisch signifikant von einem Wert β₀ abweicht.

1

Hypothesen aufstellen

Einseitig: H₀: β ≤ β₀ vs. H₁: β > β₀ Zweiseitig: H₀: β = β₀ vs. H₁: β ≠ β₀
⚠ Wichtig — Begründung der Hypothesenwahl
Wähle Hypothesen so, dass eine Testdurchführung sinnvoll ist: Die Forschungsfrage steht in H₁. Z.B. „Wir wollen zeigen, dass β > 0" → H₁: β > 0. Begründe in 1–2 Sätzen!
2

Teststatistik berechnen

T = (β̂ − β₀) / s_β̂
s² = Σε̂ᵢ² / (n−2)    (Residualvarianz) s_β̂ = s / √Σ(xᵢ−x̄)²    (Standardfehler von β̂)
3

Kritischen Wert aus t-Tabelle ablesen

Einseitig: t_{n−2; α} (Freiheitsgrade = n−2) Zweiseitig: t_{n−2; α/2}

Bei n=5 → df=3. Bei α=0,10 einseitig: t_{3;0,10} = 1,638

4

Entscheidungsregel

Einseitig (rechts): H₀ ablehnen, wenn T > t_{n−2; α} Zweiseitig: H₀ ablehnen, wenn |T| > t_{n−2; α/2}
5

Interpretation formulieren

„Auf dem X%-Signifikanzniveau wird H₀ [abgelehnt / nicht abgelehnt]. Das bedeutet, [inhaltliche Aussage über den Parameter]."

✦ Vollständiges Beispiel (angelehnt an Klausur 2024)

Aus dem KQ-Beispiel oben: β̂ = 2,0, Σε̂ᵢ² = 1.451,2, n = 5, Σ(xᵢ−x̄)² = 6.600

Teste H₀: β ≤ 0 vs. H₁: β > 0 bei α = 0,10.

s² = 1.451,2 / (5−2) = 1.451,2 / 3 = 483,73 s = √483,73 ≈ 21,99 s_β̂ = 21,99 / √6.600 = 21,99 / 81,24 ≈ 0,271
T = (2,0 − 0) / 0,271 ≈ 7,38

Kritischer Wert: t_{3; 0,10} = 1,638

Entscheidung: T = 7,38 > 1,638 → H₀ wird abgelehnt.

Interpretation: „Auf dem 10%-Niveau ist der Zusammenhang statistisch signifikant positiv. Höhere Werbeausgaben gehen mit höherem Umsatz einher."

💡 Klausur-Tipp
Begründung der Hypothesenwahl = 1–2 Sätze Pflicht! Schnurbus vergibt dafür explizit Punkte. Schreibe immer: „H₁ enthält die Forschungshypothese, da wir zeigen möchten, dass..."

⚡ Häufige Fehler

  • Freiheitsgrade vergessen: bei einfacher Regression immer df = n−2
  • Einseitig vs. zweiseitig verwechseln — lies die Aufgabe genau
  • β₀ vergessen: T = (β̂ − β₀) / s_β̂, nicht nur β̂ / s_β̂
  • Hypothesenbegründung weglassen — gibt Punktabzug
3 · Normalverteilung — Wahrscheinlichkeiten
🔴 Pflicht 📊 15–25 Pkt
1

Standardisieren

Z = (X − μ) / σ      wobei σ = √Var(X)
2

Verschiedene Wahrscheinlichkeiten umrechnen

P(X ≤ x) = Φ((x−μ)/σ) → direkt ablesen P(X > x) = 1 − Φ((x−μ)/σ) P(a < X < b) = Φ((b−μ)/σ) − Φ((a−μ)/σ) P(X = x) = 0 (stetige Verteilung!)
3

Lineare Transformationen

Wenn X₁ ~ N(μ₁, σ₁²) und X₂ ~ N(μ₂, σ₂²) unkorreliert, und Y = a·X₁ + b·X₂:

E(Y) = a·μ₁ + b·μ₂ Var(Y) = a²·σ₁² + b²·σ₂² Y ~ N(E(Y), Var(Y))
4

Quantile berechnen

x_p = μ + z_p · σ

z_p aus der Φ-Tabelle ablesen (Zeile mit Φ(z) = p), dann zurücktransformieren.

✦ Vollständiges Beispiel (aus Klausur 2026)

X₁ ~ N(3; 1,75), X₂ ~ N(−4; 1), unkorreliert, Y = 2X₁ + 3X₂

E(Y) = 2·3 + 3·(−4) = 6 − 12 = −6 Var(Y) = 4·1,75 + 9·1 = 7 + 9 = 16 SD(Y) = √16 = 4     → Y ~ N(−6; 16)

P(Y = 4) = 0, da Y stetig verteilt ist.

P(Y > 2): z = (2 − (−6)) / 4 = 8/4 = 2,00 P(Y > 2) = 1 − Φ(2,00) = 1 − 0,9772 = 0,0228
P(−0,8 < Y < 1): z₁ = (−0,8 − (−6)) / 4 = 5,2/4 = 1,30 z₂ = (1 − (−6)) / 4 = 7/4 = 1,75 P = Φ(1,75) − Φ(1,30) = 0,9599 − 0,9032 = 0,0567
90%-Quantil: z₀,₉₀ = 1,2816 y₀,₉₀ = −6 + 1,2816 · 4 = −6 + 5,13 ≈ −0,87
💡 Klausur-Tipp
P(Y = 4) = 0 bei stetiger Verteilung — das ist eine klassische Fangfrage! Kurze Begründung: „Da Y stetig verteilt ist, hat jeder Einzelwert Wahrscheinlichkeit 0." Das reicht für volle Punkte.

⚡ Häufige Fehler

  • σ² (Varianz) mit σ (Standardabweichung) verwechseln — in der Formel steht Var(Y) = a²·σ₁² mit der Varianz, nicht der SD
  • Bei negativem z-Wert: Φ(−z) = 1 − Φ(z) — Symmetrieregel benutzen
  • P(Y = x) ≠ 0 vergessen — immer „stetige Verteilung" erwähnen
  • Quantil: z aus Tabelle ablesen, dann erst zurückrechnen — nicht anders herum
4 · Korrelationskoeffizient (Bravais-Pearson)
🟠 Wichtig 📊 10–15 Pkt
Wann geeignet?
Beide Variablen metrisch + linearer Zusammenhang erwartet + keine starken Ausreißer.
1

Formel (aus Tabellenwerten)

rXY = (Σxᵢyᵢ − n·x̄·ȳ) / √[(Σxᵢ²−n·x̄²) · (Σyᵢ²−n·ȳ²)]
Alternative (konzeptuell):
rXY = Σ(xᵢ−x̄)(yᵢ−ȳ) / √[Σ(xᵢ−x̄)² · Σ(yᵢ−ȳ)²]
2

Interpretation

r = +1: perfekt positiv linear r = −1: perfekt negativ linear r = 0: kein linearer Zusammenhang |r| > 0,8: starker Zusammenhang |r| 0,5–0,8: mittlerer Zusammenhang
3

Zusammenhang mit R²

R² = rXY²

In der einfachen linearen Regression gilt: das Bestimmtheitsmaß ist das Quadrat des Korrelationskoeffizienten.

✦ Beispiel (angelehnt an Klausur 2024)

Mit den Daten aus Rezept 1: Σxᵢyᵢ = 41.850, n=5, x̄=50, ȳ=114,6, Σxᵢ²=19.100

Zusätzlich: Σyᵢ² = 40²+59²+80²+150²+244² = 1.600+3.481+6.400+22.500+59.536 = 93.517

Zähler: 41.850 − 5·50·114,6 = 13.200 Σ(xᵢ−x̄)²: 19.100 − 5·50² = 6.600 Σ(yᵢ−ȳ)²: 93.517 − 5·114,6² = 93.517 − 65.664,8 = 27.852,2 rXY = 13.200 / √(6.600 · 27.852,2) = 13.200 / √183.824.520 ≈ 13.200 / 13.558 ≈ 0,974

Interpretation: „Es besteht ein sehr starker positiver linearer Zusammenhang (r ≈ 0,974)."

⚡ Häufige Fehler

  • Σyᵢ² vergessen zu berechnen — braucht man im Nenner
  • Korrelation = Kausalität schlussfolgern — nie!
  • Vorzeichen vergessen — bei negativem Zusammenhang ist Zähler < 0
  • r = 0 mit „kein Zusammenhang" gleichsetzen — richtig ist „kein linearer Zusammenhang"
5 · Rangkorrelation (Spearman)
🟠 Wichtig 📊 10–15 Pkt
Wann statt Pearson?
Ausreißer vorhanden · nicht-linearer Zusammenhang · ordinale Daten
1

Ränge vergeben

Sortiere xᵢ und yᵢ jeweils aufsteigend und vergib Ränge 1, 2, 3, …

⚠ Bindungen (gleiche Werte)
Bei gleichen Werten: Durchschnittsrang = (Summe der betroffenen Ränge) / Anzahl. Z.B. Rang 3 und 4 geteilt → beide bekommen Rang 3,5.
2

Rangdifferenzen berechnen

dᵢ = Rang(xᵢ) − Rang(yᵢ)

Kontrolliere: Σdᵢ = 0 (wenn keine Bindungen).

3

Spearman-Koeffizient berechnen

rs = 1 − (6·Σdᵢ²) / (n·(n²−1))

Interpretation: wie Pearson, von −1 bis +1.

✦ Beispiel (aus Klausur 2023/24) — Alter und Laufzeit

n=6: Alter (x): 22, 25, 25, 30, 35, 40 · Laufzeit (y): 18, 22, 20, 25, 28, 35

Bindung bei x=25: Ränge 2 und 3 → Durchschnittsrang = 2,5 für beide.

⚡ Häufige Fehler

  • Bindungen vergessen → falscher Rang → falsche Differenzen
  • Formel mit n(n+1)(n−1) statt n(n²−1) — beides gleich, aber klar schreiben
  • Rangdifferenz dᵢ falsch berechnen — Vorzeichen spielt keine Rolle da quadriert
6 · Konfidenzintervall für β̂
🟠 Wichtig 📊 10–15 Pkt
1

Formel

KI = [β̂ − t_{n−2; α/2} · s_β̂   ;   β̂ + t_{n−2; α/2} · s_β̂]

s_β̂ wie in Rezept 2 berechnet. t-Wert immer für α/2 (zweiseitig)!

2

Interpretation

„Mit (1−α)·100 % Konfidenz liegt der wahre Parameter β in diesem Intervall." — Nicht: „mit Wahrscheinlichkeit (1−α)"!

✦ Beispiel — 90%-KI (aus den Daten in Rezept 1)

β̂ = 2,0, s_β̂ = 0,271, n=5, df=3, α=0,10 → t_{3; 0,05} = 2,353

KI = [2,0 − 2,353 · 0,271   ;   2,0 + 2,353 · 0,271] = [2,0 − 0,638   ;   2,0 + 0,638] = [1,362 ; 2,638]

„Mit 90 % Konfidenz liegt die wahre Steigung β zwischen 1,362 und 2,638."

⚡ Häufige Fehler

  • α statt α/2 beim t-Wert — für zweiseitiges KI immer α/2
  • Falsche Freiheitsgrade (nicht n, sondern n−2)
  • Interpretation mit „Wahrscheinlichkeit" formulieren — korrekt ist „Konfidenz"
7 · Bayes / Bedingte Wahrscheinlichkeit
🟠 Wichtig 📊 10–20 Pkt
1

Satz von Bayes

P(A|B) = P(B|A) · P(A) / P(B)
P(B) = P(B|A)·P(A) + P(B|Ā)·P(Ā)    (totale Wahrscheinlichkeit)
2

Baumdiagramm aufstellen

1. Ast: Urursache A und Ā mit Wahrscheinlichkeiten P(A), P(Ā).

2. Ast: Bedingte Wahrscheinlichkeiten P(B|A), P(B̄|A), P(B|Ā), P(B̄|Ā).

Endwahrscheinlichkeiten = Produkt der Äste. Alle Endwahrscheinlichkeiten summieren zu 1.

✦ Vollständiges Beispiel (aus Klausur 2024 — Spam-Filter)

Gegeben: P(F) = 0,6 (Werbung), P(F|S̄) = 0,3, P(S∩F) = 0,12

S = Spam, F = als Spam gefiltert

Aus P(S∩F) = P(F|S)·P(S) = 0,12 → benötige P(S) und P(F|S) Aus P(F) = P(F|S)·P(S) + P(F|S̄)·P(S̄) = 0,6: P(F|S)·P(S) + 0,3·(1−P(S)) = 0,6 P(S∩F) + 0,3 − 0,3·P(S) = 0,6 0,12 + 0,3 − 0,3·P(S) = 0,6 → P(S) = −0,6 / (−0,3) = 0,6 → P(F|S) = 0,12 / 0,6 = 0,2
Bayes: P(S|F) = P(F|S)·P(S) / P(F) = 0,2 · 0,6 / 0,6 = 0,2

„Die bedingte Wahrscheinlichkeit, dass eine E-Mail Spam ist, gegeben sie wurde gefiltert, beträgt 20 %."

⚡ Häufige Fehler

  • P(A|B) mit P(B|A) verwechseln — Bayes-Tausch ist der häufigste Fehler
  • P(B) nicht über totale Wahrscheinlichkeit berechnen
  • Baumdiagramm: Äste auf gleicher Stufe summieren nicht zu 1
8 · Chi-Quadrat Anpassungstest
🟡 Kann kommen 📊 10–20 Pkt
1

H₀ formulieren

„Die Variable folgt einer [Normalverteilung / Gleichverteilung / ...] mit [Parametern]."

2

Klassen und beobachtete Häufigkeiten (Oᵢ)

Aus den Daten: wie viele Beobachtungen fallen in jede Klasse?

3

Erwartete Häufigkeiten (Eᵢ) unter H₀

Eᵢ = n · P(Klasse i unter H₀)

P(Klasse i) aus Normalverteilung: P(a < X < b) mit gegebenen μ, σ berechnen → standardisieren.

4

Teststatistik berechnen

χ² = Σ (Oᵢ − Eᵢ)² / Eᵢ
5

Freiheitsgrade

df = k − 1 − (Anzahl geschätzter Parameter)

Normalverteilung mit μ und σ aus Daten geschätzt: df = k − 1 − 2 = k − 3

6

Entscheidung

H₀ ablehnen, wenn χ² > χ²_{df; α} aus der Tabelle.

✦ Beispiel (aus Klausur 2026) — Händler, 50 Produkte

Klassen: [0,20), [20,40), [40,60] · n=50 · μ=30 (Mitte), σ=√183,67≈13,55 · α=0,05

P([0,20)): z₁=(0−30)/13,55=−2,21, z₂=(20−30)/13,55=−0,74 P = Φ(−0,74)−Φ(−2,21) = 0,2296−0,0136 = 0,2160 E₁ = 50·0,2160 = 10,8 P([20,40)): z₁=−0,74, z₂=(40−30)/13,55=0,74 P = Φ(0,74)−Φ(−0,74) = 0,7704−0,2296 = 0,5408 E₂ = 50·0,5408 = 27,04 P([40,60]): z₁=0,74, z₂=(60−30)/13,55=2,21 E₃ = 50·0,2160 = 10,8

O = (8, 30, 12), E = (10,8, 27,04, 10,8)

χ² = (8−10,8)²/10,8 + (30−27,04)²/27,04 + (12−10,8)²/10,8 = 7,84/10,8 + 8,76/27,04 + 1,44/10,8 = 0,726 + 0,324 + 0,133 ≈ 1,18

df = 3 − 1 − 2 = 0 → Achtung: df=0 bedeutet Test nicht anwendbar (zu wenig Klassen für die geschätzten Parameter). Schnurbus gibt in solchen Fällen df explizit vor.

⚡ Häufige Fehler

  • Freiheitsgrade falsch: df = k − 1 − Anzahl geschätzter Parameter
  • Klassen mit Eᵢ < 5 sollten zusammengefasst werden
  • Standardisierung vergessen: μ und σ immer angeben
9 · Binomialverteilung
🟡 Kann kommen 📊 10–15 Pkt
1

Wahrscheinlichkeitsfunktion

P(X = k) = C(n,k) · πᵏ · (1−π)^(n−k)
C(n,k) = n! / (k! · (n−k)!)
2

Kenngrößen

E(X) = n·π     Var(X) = n·π·(1−π)
3

Kumulierte Wahrscheinlichkeiten

P(X ≤ k) = Σᵢ₌₀ᵏ P(X=i) P(X > k) = 1 − P(X ≤ k)
✦ Beispiel — Elfmeter, n=5, π=0,6
P(X=0) = C(5,0)·0,6⁰·0,4⁵ = 1·1·0,01024 = 0,010 P(X=1) = C(5,1)·0,6¹·0,4⁴ = 5·0,6·0,0256 = 0,077 P(X=2) = C(5,2)·0,6²·0,4³ = 10·0,36·0,064 = 0,230 P(X=3) = C(5,3)·0,6³·0,4² = 10·0,216·0,16 = 0,346 P(X=4) = C(5,4)·0,6⁴·0,4¹ = 5·0,1296·0,4 = 0,259 P(X=5) = C(5,5)·0,6⁵·0,4⁰ = 1·0,07776·1 = 0,078 E(X) = 5·0,6 = 3    Var(X) = 5·0,6·0,4 = 1,2 P(X≥3) = P(3)+P(4)+P(5) = 0,346+0,259+0,078 = 0,683

⚡ Häufige Fehler

  • C(n,k) falsch berechnen — bei großem n den Taschenrechner benutzen
  • P(X > k) = P(X ≥ k+1), nicht P(X ≥ k) — Grenzen genau lesen
10 · Varianzanalyse (ANOVA)
🟡 Kann kommen 📊 15–20 Pkt
Wann?
Mittelwerte von mehr als zwei Gruppen vergleichen · H₀: μ₁ = μ₂ = ... = μₖ
1

Streuungszerlegung

SST = SSB + SSW SST = Σᵢⱼ (yᵢⱼ − ȳ)² (Gesamtstreuung) SSB = Σⱼ nⱼ·(ȳⱼ − ȳ)² (Zwischen-Gruppen) SSW = Σᵢⱼ (yᵢⱼ − ȳⱼ)² (Innerhalb-Gruppen)
2

F-Statistik

F = (SSB/(k−1)) / (SSW/(n−k))    = MSB / MSW

k = Anzahl Gruppen, n = Gesamtbeobachtungen

3

Entscheidung

H₀ ablehnen, wenn F > F_{k−1; n−k; α} aus der F-Tabelle.

⚡ Häufige Fehler

  • Freiheitsgrade: Zähler = k−1, Nenner = n−k
